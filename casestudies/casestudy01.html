CASE STUDIES - STAR-CCM

Running STAR-CCM+ on Azure Cloud

"Excellent performance of the STAR-CCM+ container eliminates the need to have all software in-house. HPC on Azure provides the packaged software ready to go for faster analysis."


The intake manifold of an automotive engine is a critical system as it relates directly to the engineâ€™s efficiency. When designing automotive engines, it is important to use flow analysis using CFD simulation software to achieve efficient designs and economical manifolds.


To test this in the cloud, CAE Technology Inc ran a V6 intake manifold simulation on Microsoft Azure to experiment with performance. Using UberCloud Application Containers that ran STAR-CCM+ on a cluster of Microsoft Azure A9 instances, totaling 128 cores and 1TB of RAM, CAE Technology Inc ran the simulation for 10,000 iterations. The simulation was highly successful. The UberCloud container together with Azure compute instances produced high computational speed, taking 18708 seconds (5h 20m) to run 10,000 iterations.


CAE Technology Inc found that running on the 128 cores on Microsoft Azure was extremely fast compared to their in-house alternatives, and found the overall experience with Azure HPC Cluster to be intuitive and easy to use.
