Why work on Azure?


World Class Compute

Run your work using high-end compute nodes with QDR Infiniband, and take advantage of a backend network with MPI latency of under 3 micro-seconds and non-blocking 32 GB/second throughput.

Stay focused on design,
we’ll take care of the rest

We’ll provide the infrastructure so that you can spend more time honing in on the right design, less time getting your products to the market.

Burst To Cloud

Run out of compute? Our seamless Enterprise-ready solutions can help you burst in to the cloud, letting you scale your resources as need. On Azure, you pay only for what you’re using.

Work faster, save more

With our high performance hardware and the scalability of Azure, you’ll be able to run more simulation in less time. With increased efficiency, you’ll make the most out of your software licenses, getting more done with lower costs.

Network optimized: fast networking with InifiniBand support

The A8 and A9 virtual machines feature Intel® Xeon® E5 processors, with 32 Gbit/s Infiniband network and remote direct memory access (RDMA) technology. These instances are ideal for Message Passing Interface (MPI) applications, high-performance clusters for modeling and simulations, video encoding, and other compute or network intensive scenarios. These instances are available in select data centers.
